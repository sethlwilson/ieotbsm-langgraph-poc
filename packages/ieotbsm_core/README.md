# ieotbsm-core

Pure-Python trust engine extracted from the IEOTBSM PoC: inter-organizational ledger (τ), ISP-style `TrustGate`, TPM modes, and provenance structures. Pydantic schemas in `ieotbsm_core.schemas` are the stable boundary for HTTP/MCP services.

Install from repo root:

```bash
pip install -e ./packages/ieotbsm_core
```
